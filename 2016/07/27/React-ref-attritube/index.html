React ref Callback Attribute

Jul 27, 2016 | Hits

套用framework時，看到一個不懂的地方，下面的code ref={node => (this.root = node)}，在以前的經驗，該ref屬性都是給字串，但這回變成給 function 了，丫丫，突然變笨了，怎麼會這樣…。

當然只能問一下原廠的文件，果然也看不懂，再問了google後，大約可以猜得出來這段code的用意了。

在render()時，ref={node => (this.root = node)}的意思是，把自己的ref傳給this.root，然後在render()完後就會跑componentDidMount() (react component lifecycle)，此時的this.root就是 <div className="mdl-layout mdl-js-layout" />，然後再給mdl作upgradeElement嚕，這樣mdl的元件就可以正常運作了。
